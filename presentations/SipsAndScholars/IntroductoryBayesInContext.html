\[\begin{align} Y_i\mid \mu &\substack{\text{ind}\\ \sim} N\left(\mu, \sigma^2\right)\\ \mu &\sim N\left(\theta, \tau^2\right)\\ \mu \mid \vec{y} &\sim N\left(\theta\frac{\sigma^2}{n\tau^2 + \sigma^2} + \bar{y} \frac{n\tau^2}{n\tau^2 + \sigma^2}, \frac{\tau^2\sigma^2}{n\tau^2 + \sigma^2}\right) \end{align}\]

gibbs_normal <- function(input, S, seed){
  set.seed(seed)
  ybar <- mean(input$y)
  n <- length(input$y)
  para <- matrix(0, S, 2)
  phi <- input$phi_init
  for(s in 1:S){
    theta1 <- (input$theta/input$tau^2 + n*phi*ybar)/
    (1/input$tau^2 + n*phi)
    tau1 <- sqrt(1/(1/input$tau^2 + n*phi))
    mu <- rnorm(1, mean = theta1, sd = tau1)
    alpha1 <- input$alpha + n/2
    beta1 <- input$beta + sum((input$y - mu)^2)/2 
    phi <- rgamma(1, shape = alpha1, rate = beta1)
    para[s, ] <- c(mu, phi)
  }
  return(para)
}

Why Bayes Is Still Worth It

  • Humans already intuitively think in Bayesian ways
  • Bayesian thinking aligns with how we interact with the world around us
  • Bayesian updating is transparent and simulation-friendly
  • Coding has gotten much easier and is broadly accessible
  • Interpretations resulting from Bayesian approaches are much more natural than from the other methods

Summary: While the details are complex and technical, the process and how to interpret results is intuitive

Project Goals

Create no/low-barrier, context-flexible activities that:

  • Expose students to foundational Bayesian ideas within their home disciplines

  • Are adaptable to applications meaningful to students in any discipline

  • Are easy for faculty to adopt or tailor

  • Require no technical setup for class use

  • Support several Bayesian skills:

    • Parameter estimation
    • Hypothesis testing
    • Introductory modeling

Introductory Bayes in Context

A set of modular, context-free Bayesian statistics activities

  • providing a framework for overlaying any context
  • so faculty can give students discipline-relevant Bayesian experiences without needing to design a full workflow from scratch.

Sample Activity: Chytrid Fungal Infection in Frogs (with Katie Duryea)

  • Authentic story and data from biology
  • Real-world ecological implications
  • Natural framing for uncertainty and updating

Students can:

  • Explore disease prevalence
  • Make and explore their own choices and beliefs (priors)
  • Adjust priors based on frogs observed and processed by SNHU students and faculty
  • Interpret posterior changes in terms of biological meaning

How It All Works

  • Foundational Bayesian and statistics workflows are unchanged – use data to update our prior assumptions about a population parameter.

  • The context overlay is completely customizable though…

    • Limited pre-reading about the context and method
    • Images, videos, and other embedded media
    • Discipline-specific background and objectives to keep content coverage moving along
    • Discipline-relevant justifications, interpretations, and more
    • Adaptations to question framing

  • The result is a single activity with many possible discipline-specific, and truly relevant versions

How It Works (In Practice)

  1. Identify the discipline-specific story
  2. Swap in:
    • A few short text blocks
    • The contextual images, videos, or other media
    • A small set of prompts relevant to your class objectives and students
  3. Publish using GitHub Pages
